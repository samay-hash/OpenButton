## 🚀 Quick Start

Get your environment running locally in under 60 seconds.

### 1. Clone & Install
```bash
git clone https://github.com/samay-hash/openbutton.git
cd openbutton
pnpm install
```

### 2. Ignite Development Server
```bash
pnpm run dev
```
Navigate to <kbd>http://localhost:3000</kbd> to experience the UI architecture firsthand.
